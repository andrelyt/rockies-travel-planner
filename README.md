# Rockies Travel Planner

加拿大落基山、温哥华与西雅图 11 日交互式旅行路书。

## 在线查看

https://andrelyt.github.io/rockies-travel-planner/

## 本地运行

```bash
npm install
npm run dev
```

## 数据来源

- 地图：OpenFreeMap / OpenMapTiles / OpenStreetMap
- 天气：Open-Meteo
- 景点图片：Wikimedia Commons，图片已保存到本项目；许可信息见 `public/images/ATTRIBUTION.md`
